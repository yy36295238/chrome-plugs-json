# Chrome插件,JSON格式化、对比工具

> 这是一个Chrome插件，用来操作JSON数据的

### 特征
1. 支持JSON格式化
2. 支持JSON多分隔栏显示
3. 支持JSON比较
4. 支持演示数据
5. 支持JSON清除数据
6. 支持展开合并
7. 支持JSON数据合并

### 使用方法
1. 下载：将代码下载到本地
2. 打开：Chrome浏览器并转到：chrome://extensions/
3. 启用："开发者模式"
4. 点击："加载已解压的扩展程序"
5. 选择：选择刚下载好的文件目录。
